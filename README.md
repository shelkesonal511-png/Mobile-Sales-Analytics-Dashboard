# 📊 Mobile Sales Analytics Dashboard

## 🚀 Overview
This project presents an **interactive Mobile Sales Dashboard** built using **Microsoft Power BI** to transform raw sales data into actionable insights.  

It helps analyze **sales performance, customer behavior, and market trends** across different cities, brands, and time periods.

---

## 🎯 Objectives
- Analyze overall mobile sales performance  
- Identify top-performing cities and brands  
- Understand customer purchasing behavior  
- Track daily and monthly sales trends  
- Compare different payment methods  

---

## 📌 Key Features
- 💰 **Total Sales:** 769M+  
- 📦 **Total Quantity Sold:** 19K units  
- 💳 **Payment Analysis:** UPI, Debit Card, Cash  
- 🌍 **City-wise Sales Map Visualization**  
- 📅 **Daily Sales Trend Analysis**  
- ⭐ **Customer Ratings Insights**  
- 📱 **Brand & Model Performance Comparison**  
- 🎛️ Interactive filters (Month, Brand, Model, Payment Method, Day)
